Inserindo um Formulário em uma página web

Elementos práticos

Criar a estrutura básica da página HTML e o container do formulário, contendo campos para a inserção de:

  • Nome;
  • E-mail;
  • Telefone de contato;
  • Calendário;
  • Data de nascimento;
  • Inserção de texto livre;
  • Ao final, teste no navegador

Motivação

  • As páginas HTML são muito populares e intuitivas;
  • Velocidade de desenvolvimento;
  • Acelerar a capacidade de obter informações.
