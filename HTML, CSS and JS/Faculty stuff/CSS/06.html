PseudoClasses

Pseudoclasses são uma forma de estilizar elementos com base em seu estado ou posição. Por exemplo, a pseudoclasse :hover é usada para aplicar estilos quando o mouse passa sobre um elemento.

Passe o mouse sobre este texto para ver a pseudoclasse :hover em ação.

PseudoElementos

Pseudoelementos são usados para estilizar partes específicas de um elemento. Por exemplo, o pseudoelemento ::first-line é usado para aplicar estilos à primeira linha de um elemento. Como você pode ter notado, neste texto, a primeira linha esta em letras maísculas na cor azul.
